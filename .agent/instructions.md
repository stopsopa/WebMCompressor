# Project Orchestration Rules

You MUST strictly follow the specialized skills located in `.agent/skills/`. Before performing any work in their respective domains, read the corresponding `SKILL.md` file using `view_file`.

## 1. Shell Scripting

- **Skill**: `bash-scripting` (`.agent/skills/bash-scripting/SKILL.md`)
- **When to use**: Whenever creating or modifying `.sh` files.

## 2. Infrastructure & Docker

- **Skill**: `docker-coding-style` (`.agent/skills/docker-coding-style/SKILL.md`)
- **When to use**: Whenever working with `docker-compose.yml`, Dockerfiles, or infrastructure scripts in `docker/`.

## 3. TypeScript & Runtime

- **Skill**: `typescript-node-native-rules` (`.agent/skills/typescript-node-native-rules/SKILL.md`)
- **When to use**: For all TypeScript development and Node.js execution logic.
